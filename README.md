# Travel Order Resolver

## Install dependencies

### On Linux
`sudo apt-get install python-pyaudio python3-pyaudio`  
`pip install SpeechRecognition`

### On MacOS (M1)
`brew install portaudio`  
`pip install pyaudio`  
`brew install flac`  
`pip install SpeechRecognition`

### On Windows
`pip install pyaudio`  
`pip install SpeechRecognition`
